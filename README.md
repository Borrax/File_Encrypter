# File_Encrypter

![rust logo](https://rust-lang.org/logos/rust-logo-512x512.png)

### General Information:
Encrypts a file or multiple files by a given path using the AES-265-GCM algorithm.

### Usage:
```bash
cargo run -- <file_path>
```
